# Customer Segmentation and Churn Prediction

A Machine Learning project that segments customers using **K-Means Clustering** and predicts customer churn using classification models. The project helps identify customer groups, estimate churn probability, and provide insights for improving customer retention.

## Features
- Data Preprocessing
- Exploratory Data Analysis (EDA)
- Customer Segmentation using K-Means
- Customer Churn Prediction
- Data Visualization
- Business Insights

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn
- Jupyter Notebook

## Project Workflow
1. Data Preprocessing
2. Exploratory Data Analysis
3. Feature Scaling
4. K-Means Clustering
5. Churn Prediction
6. Data Visualization

## Results
- Segmented customers into distinct groups.
- Predicted customer churn probability.
- Generated insights to support customer retention strategies.
